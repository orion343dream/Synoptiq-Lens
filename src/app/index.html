The Integration: ID Trace

One ID. Universal Coverage. Zero Hassle.

Integrating Synoptic Lens is simple and secure. We bypass complex configuration by using a single, unique ID generated for your application, allowing us to securely ingest critical metadata and log streams.

  • 1. Generate Secure ID: Create a unique, encrypted tracking key in the Synoptic Platform.
  • 2. Embed Snippet/API: Insert the generated ID into your application's configuration or a small integration snippet.
  • 3. Secure Connection: The ID enables a secure, read-only connection to relevant API endpoints and database logs.
  • 4. Instant Visualization: Your data immediately begins flowing into the Synoptic Lens dashboard for analysis.
